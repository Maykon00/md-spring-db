# md-spring-db

## Description
Simple CRUD project using H2 in-memory database.

## Technologies
- Java 21
- Spring Boot 3.5.0
- Maven
- H2 Database
- JPA
- Swagger (OpenAPI) for API documentation

## Requirements
- Java 21 installed
- Maven 3.9.x or higher installed

## Build and Run

### Build the project
`mvn clean install`
### Run the project
`mvn spring-boot:run`
